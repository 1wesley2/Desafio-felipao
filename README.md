# Desafio Felipão DIO

Este repositório contém o script de configuração do `modo study` utilizado pelo assistente Copilot durante o desenvolvimento. É um material de apoio para estudos de lógica de programação em JavaScript e Node.js.

## Conteúdos
- `modo-study.md`: arquivo copiado das configurações do VS Code contendo instruções do modo study.
- Eventuais exemplos e exercícios em Node.js.

## Uso
Basta clonar o repositório e ler o arquivo `modo-study.md`. Opcionalmente, rode os scripts de exemplo com Node:
```bash
node ex1.js
```
